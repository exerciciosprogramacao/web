📘 Documentação do Projeto - Cadastro de Pessoas

🎯 Objetivo do Projeto

- Criar um sistema de cadastro de pessoas, onde informações como nome, e-mail e idade são inseridas em um formulário.
- Cada nova pessoa cadastrada será exibida em um "card" na tela.
- O formulário deve ser limpo após o envio e os dados armazenados em uma lista.

🖱️ 3. Detectando o Envio do Formulário

// Adicionando um ouvinte de evento para o envio do formulário
[📮].addEventListener('[...]', function([🎲]) {
  // Aqui dentro você programará o que acontece ao enviar o formulário.
});

👉 O addEventListener é usado para escutar o evento de envio do formulário. O parâmetro [🎲] normalmente é chamado de event e representa o evento que queremos capturar (neste caso, submit).

❌ 4. Prevenindo o Comportamento Padrão

// Prevenindo que o formulário recarregue a página
[🎲].preventDefault();

👉 O preventDefault evita que a página seja recarregada ao enviar o formulário, algo que acontece por padrão com o evento submit.

🔍 8. Função de Exibir os Cards

function exibirCards(){
  [📦].innerHTML = '';
  
  // Para cada pessoa, criamos um card
  [📑].forEach([🔌] => {
    const [🎴] = document.createElement('div');
    [🎴].classList.add('[......]');
    [🎴].innerHTML = 
          "<h3>" + [🔌].nome + "</h3>" +
          "<p>" Email:  + [🔌].email + "</p>" +
          "<p>" Idade:  + [🔌].idade + "</p>";
  
    
    // Adicionando o card à área de exibição
    [📦].appendChild([🎴]);
  });
}

👉 Aqui, usamos forEach para percorrer a lista de pessoas [📑] e criar um card para cada item da lista. Cada card é adicionado ao container de cards [📦].
