📘 Documentação do Projeto - Carrossel

🎯 Objetivo do Projeto

- Criar um carrossel de imagens funcional usando JavaScript puro.
- Ao clicar em um botão, a imagem correspondente deve aparecer.
- Somente uma imagem e um botão ficam ativos por vez.

👉 Um exercício de lógica e manipulação de DOM com interações visuais.

👉 Você vai reparar que existem [] ou [......] "lacunas" nos exemplos de códigos abaixo, essas lacunas devem ser preenchidas.

👉 Elas servem para que você consiga colocar em prática conceitos já aprendidos ou que seram mostrados.

👉 Em alguns casos teremos ícones, eles são para você ver que por exemplo: a mesma variável vai ser usado em outra parte então, o ícone vai estar em outro lugar do código.

🧱 1. Selecionando os Elementos

👉 Vamos usar uma lógica muito similar a que usamos no exercício do card com javaScript.

// Como selecionar todos os botões do carrossel?
const [📮] = document.query[......]('[......]');
  
// Agora selecione todas as imagens do carrossel.
const [📦] = document.query[......]('[......]');

👉 query... retorna uma lista de todos os elementos que possuem a classe informada. Essa lista permite que você acesse cada botão ou imagem pelo índice (como se fosse um array).

👉 Exemplo: document.query...('.icones') seleciona todos os ícones com a classe ícones.

🖱️ 2. Detectando Cliques nos Botões

// Use forEach para percorrer os botões
[📮].forEach(([🎲], [📚]) => {
  [🎲].add[......]('[🖱]', () => {
    // Aqui dentro você programará o que acontece ao clicar.
  });
});

👉 forEach é um método usado para percorrer listas (como as que vêm do querySelectorAll). Para cada botão, ele executa uma função.

👉 add... é usado para reagir a eventos. Neste caso, estamos ouvindo o evento `click` em cada botão.

👉 Quando o botão for clicado, a função dentro do `addEventListener` será executada.

❌ 3. Desmarcando o Botão Anterior

// Dentro do clique:
const [📪] = document.querySelector('.[......]');
[📪].classList.remove('[......]');

👉 querySelector retorna o primeiro elemento que possui a classe informada. Aqui, ele pega o botão que está atualmente com a classe selecionado.

👉 Usamos classList.remove() para tirar a classe selecionado dele. Repare que não temos o . "ponto" antes da classe pois nos referimos a classe em sí e não ao seletor.

👉 Quando falamos de seletores temos sempre o . "ponto" na frente, quando falamos de classe não temos o ponto.

✅ 4. Marcando o Novo Botão

// Adicione a classe "selecionado" no botão que foi clicado.
[🎲].classList.add('[......]');

👉 O botão clicado está representado pela variável. Ao usar classList.add('[......]'), ele ganha o estilo visual de selecionado.

🧼 5. Escondendo a Imagem Atual

// Selecione a imagem atual que está com a classe "ativa"
const [🧰] = document.querySelector('.[......]');
[🧰].classList.remove('[......]');

👉 Assim como fizemos com os botões, removemos a classe da imagem atual, que está sendo exibida.

👁️ 6. Mostrando a Imagem Certa

// Use o índice do botão clicado para acessar a imagem correspondente
[📦][[📚]].classList.add('[......]');

👉 A imagem correta está na mesma posição (índice) que o botão é clicado.

👉 Assim, usamos [📦][[📚]] para pegar a imagem certa e classList.add('[...]') para exibi-la.
