📘 Documentação do Projeto - Clicker

🎯 Objetivo do Jogo

- Clique na imagem para ganhar moedas.
- Use as moedas para comprar melhorias:
  • Mais moedas por clique
  • Mais moedas por segundo

👉 Um jogo simples onde o jogador evolui clicando e comprando upgrades.

🧱 1. Selecionando os Elementos da Página

const [📷] = document.querySelector('[.....]');
const [📊] = document.querySelector('[.....]');
const [⏳] = document.querySelector('[.....]');
const [💰] = document.querySelector('[.....]');

👉 Esses comandos pegam os elementos HTML que serão atualizados no jogo. [📷] seleciona a imagem, [📊] é o contador de moedas, [⏳] é o contador de segundos e [💰] é o contador de cliques.

⚙️ 2. Criando Variáveis Iniciais

let [💸] = 1;
let [⏱️] = 0;
let [📮] = 0;

👉 Variáveis para controlar cliques, produção automática e total de moedas. [💸] é o número de cliques por vez, [⏱️] é o número de segundos de produção automática e [📮] é o total de moedas.

🔄 3. Atualizando os Valores na Tela

[📊].innerText = "Moedas: " + [📮];
[💰].innerText = "Cliques: " + [💸];

👉 Exibe as moedas e o valor de cada clique logo ao iniciar o jogo. Os elementos da página são atualizados com os valores das variáveis [📮] e [💸].
